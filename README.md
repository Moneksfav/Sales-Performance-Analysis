# Sales-Performance-Analysis
Using Excel for both Data Cleaning and interactive visualization, this is an analysis of sales performance across different regions and its monthly profit trends based on category. I made use of Power Query for the heavy lifting (data cleaning), Pivot Tables & Calculated Fields for the logic, Key performance indicators(KPI) to monitor sales and its profit, and finally, slicers for an interactive, user-friendly UI
